Secuencia

Secuencia is a jQuery plugin that allows user to manually control sequences.

Note: The only required argument in the Secuencia plugin is the number of frames that your sequence has.

Simplest Secuencia

Select your sequence element: $('#dog-0').secuencia({nFrames: 6});

Vertical animation

Select your sequence element: $('#dog-1').secuencia({direction: 'vertical', nFrames: 6});

Increase speed and change listener

Select your sequence element: $('#dog-2').secuencia({listener: "#dog-area-2", nFrames: 6, speed: 2});
